Kennard-Stone algorithm and SPXY algorithm and python implementation

Kennard-Stone (KS) and SPXY algorithms are common sample split algorithms in the chemometrics field.. Unlike random split, these two methods are based on the so called "maximum minimum distance", which means the split result of KS and SPXY are definite instead of random. This article will briefly introduce these 2 algorithms and provide a python implementation.
